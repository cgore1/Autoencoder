neural_net.py -

Contains implementation of 2 hidden layer neural network.
(Structure of neural network can be changed by changing layerNodes array)

Takes input from Trainimages for training and input from TestImages directory for testing.

The network can be tested by calling test() on the network.
The output is stored in directory test_rec.
The visualization of weights (task 3) is stored in Weights folder
